# 项目管理
项目是系统的核心

## 基本信息管理
- 甲方
    - 名称
    - 联系人
    - 电话
- 项目名称
- 承接日期
- 项目概况
- 出图单位
- 项目编号

## 经营管理

### 经营人员配置
- 经营负责人
- 经营参与人

### 招投标阶段
- 招标文件	
- 投标文件	
- 中标通知书	
- 专家费支付（现金/转账、金额、人员）

### 合同信息
- 合同文件
    - 签订时间
    - 合同金额（类型：设计费、勘察费、咨询费（元））
    - 合同费率
- 补充协议文件
   - 签订时间
    - 合同金额（类型：设计费、勘察费、咨询费（元））
    - 合同费率


### 甲方支付
- 甲方支付
    - 时间
    - 金额

- 我方开票
    - 时间
    - 金额
    - 发票文件


### 奖金分配
   - 发放人员
    - 发放金额
    - 发放时间

### 经营信息统计
- 总应收金额（合同金额）
- 已收金额（甲方支付）
- 未收金额（应收减去已收）


## 生产管理

### 生产人员配置
- 生产负责人
- 生产人员
- 审核人
- 审定人

### 项目批复、审计阶段
- 批复报告
- 批复金额：设计费、勘察费、咨询费（元）
- 审计报告	
- 审计金额：设计费、勘察费、咨询费（元）

相关文件需要上传

### 生产阶段

1. 方案
2. 初步设计
3. 施工图设计
4. 变更洽商
5. 竣工验收

每个步骤都有文件产出，其中1、2、3额外需要校审单和评分

### 生产信息统计
- 生产成本信息
    - 打车（时间、里程、金额）
    - 住宿（时间、金额）
    - 公共交通（时间、金额）
需要发票

- 对外委托及支付信息
    - 委托类型（个人或单位）
    - 对委托方的评分
    - 委托合同
    - 给委托方支付（金额、时间、索要发票）
- 奖金信息
    - 发放人员
    - 发放金额
    - 发放时间


## 项目人员管理
项目人员是项目配置中的一环，项目人员不一定有账号
不需要为单个项目单独配置权限，权限对所有项目都一致
### 项目管理员
- 拥有项目管理权限才能创建项目

### 经营人员配置
- 经营负责人，主要负责经营信息管理
- 经营人员（无账号，不能使用系统）

### 生产人员配置
- 生产负责人，主要负责生产信息管理
- 生产人员（无账号，不能使用系统）
    - 分不同专业的设计人（例如道路专业、桥梁专业，可自定义配置专业）
    - 分不同专业的参与人
    - 分不同专业的复核人
- 审核人（无账号，不能使用系统）
- 审定人（无账号，不能使用系统）



# 公司收入管理（单个项目维度）

需要权限隔离，配置了才能查看公司收入

- 管理费设置
- 信息汇总
    - 总应收金额（设计费、勘察费、咨询费（元））
    - 所有发票信息
    - 所有支付信息
    - 未收金额
