# 生产成本管理功能规格

**模块**: project-management/production  
**状态**: 🟡 开发中  
**最后更新**: 2026-03-17

## 概述

生产负责人能够记录项目的生产成本信息，包括打车、住宿、公共交通等费用，并上传相关发票。准确的成本记录有助于项目利润分析和成本管理。

## 用户故事

### US-20 记录生产成本 (Priority: P2)

**验收场景**：

1. **Given** 项目产生生产成本，**When** 生产负责人记录成本信息（打车：时间、里程、金额；住宿：时间、金额；公共交通：时间、金额）并上传发票，**Then** 系统保存成本信息并计算成本统计
2. **Given** 成本记录已存在，**When** 生产负责人编辑或删除成本记录，**Then** 系统更新或删除成本记录并重新计算成本统计
3. **Given** 用户需要查看成本统计，**When** 用户查看项目生产信息，**Then** 系统显示所有生产成本记录及成本统计

## 功能需求

- **FR-016**: 系统必须支持生产成本信息管理，包括打车（时间、里程、金额）、住宿（时间、金额）、公共交通（时间、金额）的记录，以及相关发票的上传

## 边界情况

- **EC-030**: 成本发票上传失败：成本发票上传失败时，系统提示错误信息，允许用户重新上传，已录入的成本信息（时间、金额等）保留

## 成本类型

| 类型 | 字段 |
|------|------|
| 打车 | 时间、里程、金额、发票 |
| 住宿 | 时间、金额、发票 |
| 公共交通 | 时间、金额、发票 |

## 关键实体

- **财务记录（FinancialRecord）**：生产成本通过财务记录实体管理，类型区分打车/住宿/公共交通
