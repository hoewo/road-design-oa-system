2. 我们收集的信息

2. Information We Collect

2.1 不收集任何个人信息

2.1 No Personal Information Collection

本应用不收集、不存储、不上传以下任何信息：

This app does not collect, store, or upload any of the following information:

2.2 本地数据存储

2.2 Local Data Storage

本应用仅在您的设备本地存储以下信息：

This app only stores the following information locally on your device:

  • 照片评分数据：当您为照片设置评分（0-5星）时，评分信息会同时以XMP（可扩展元数据平台）和EXIF两种标准格式写入到照片文件的元数据中。XMP格式用于专业照片管理软件的兼容性，EXIF格式提供更广泛的兼容性支持。此数据完全保存在您的本地设备上，不会离开您的设备。
  • Photo Rating Data: When you set a rating (0-5 stars) for a photo, the rating information is written to the photo file's metadata in both XMP (Extensible Metadata Platform) and EXIF standard formats. XMP format is used for compatibility with professional photo management software, while EXIF format provides broader compatibility. This data is completely stored on your local device and never leaves your device.
  • 用户偏好设置：包括排序方式、窗口大小等界面偏好设置，这些设置仅存储在您的Mac系统UserDefaults中，不会与任何第三方共享。
  • User Preferences: Including sorting methods, window sizes, and other interface preferences. These settings are only stored in your Mac's UserDefaults and are not shared with any third parties.

3. 文件访问权限

3. File Access Permissions

3.1 权限范围

3.1 Permission Scope

本应用运行在macOS沙盒环境中，具有严格的文件访问限制：

This app runs in a macOS sandbox environment with strict file access restrictions:

  • 应用只能访问您明确授权选择的文件夹
  • The app can only access folders that you explicitly authorize and select
  • 应用使用 com.apple.security.files.bookmarks.app-scope 权限，确保只能访问您指定的文件夹
  • The app uses the com.apple.security.files.bookmarks.app-scope permission to ensure it can only access folders you specify
  • 应用无法访问您未明确选择的任何其他文件夹或文件
  • The app cannot access any other folders or files you have not explicitly selected
  • 应用无法访问系统目录或其他应用的私有数据
  • The app cannot access system directories or private data of other applications

3.2 文件操作

3.2 File Operations

本应用对您选择的文件夹中的图片文件进行以下操作：

This app performs the following operations on image files in the folder you select:

  • 读取操作：读取图片文件用于显示和预览
  • Read Operations: Reading image files for display and preview
  • 元数据读取：读取图片文件中的EXIF和XMP元数据，特别是评分信息
  • Metadata Reading: Reading EXIF and XMP metadata from image files, particularly rating information
  • 元数据写入：当您设置照片评分时，将评分信息写入到图片文件的XMP和EXIF元数据中
  • Metadata Writing: When you set a photo rating, writing the rating information to the image file's XMP and EXIF metadata
  • 删除操作：应用提供删除功能，但仅在您主动点击删除按钮时才会执行。删除操作会将照片文件移动到macOS系统垃圾桶（废纸篓），您可以从垃圾桶中恢复文件。应用不会自动删除您的照片文件。
  • Delete Operations: The app provides delete functionality, but it only executes when you actively click the delete button. Delete operations move photo files to the macOS system Trash, from which you can recover files. The app does not automatically delete your photo files.
  • 应用不会自动移动或复制您的照片文件
  • The app does not automatically move or copy your photo files
  • 应用不会修改照片的像素内容，仅修改元数据
  • The app does not modify the pixel content of photos, only metadata
安全说明：Security Note: 当您首次选择文件夹时，macOS会向您请求访问权限。您可以随时在"系统设置" > "隐私与安全性" > "文件和文件夹"中撤销此权限。 When you first select a folder, macOS will request access permission. You can revoke this permission at any time in "System Settings" > "Privacy & Security" > "Files and Folders".

4. 数据使用方式

4. How We Use Data

4.1 本地处理

4.1 Local Processing

本应用的所有数据处理均在您的Mac设备本地完成：

All data processing of this app is completed locally on your Mac device:

  • 图片加载和显示在本地进行
  • Image loading and display are performed locally
  • 搜索、过滤和排序功能使用本地算法处理
  • Search, filter, and sort functions use local algorithms
  • 缩略图生成和缓存存储在本地
  • Thumbnail generation and caching are stored locally
  • 评分数据的读取和写入在本地文件系统中完成
  • Reading and writing of rating data is completed in the local file system

4.2 功能说明

4.2 Feature Description

本应用提供以下功能，所有功能均在本地执行：

This app provides the following features, all executed locally:

  • 图片浏览：支持JPG、PNG、GIF、BMP、TIFF、WebP以及多种RAW格式（CR2、CR3、NEF、ARW、DNG、RAF、ORF等）
  • Image Browsing: Supports JPG, PNG, GIF, BMP, TIFF, WebP, and various RAW formats (CR2, CR3, NEF, ARW, DNG, RAF, ORF, etc.)
  • 图片预览：支持缩放、拖拽查看等交互功能
  • Image Preview: Supports interactive features such as zooming and dragging
  • 照片评分：为照片设置0-5星评分，评分数据同时存储在照片文件的XMP和EXIF元数据中，确保与其他照片管理软件的兼容性
  • Photo Rating: Set 0-5 star ratings for photos. Rating data is stored in both XMP and EXIF metadata of photo files, ensuring compatibility with other photo management software
  • 搜索过滤：按文件名搜索、按评分过滤、按名称/日期/评分排序
  • Search and Filter: Search by filename, filter by rating, sort by name/date/rating
