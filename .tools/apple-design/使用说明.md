# 设计令牌展示页面使用说明

## 📱 如何查看设计令牌展示页面

已将设计令牌相关文件复制到 Xcode 项目中，你可以通过以下方式查看：

### 方式 1：在 UICopyApp.swift 中切换（推荐）

1. 打开 `UICopy/UICopyApp.swift`
2. 找到 `WindowGroup` 部分
3. 注释/取消注释来切换页面：

```swift
var body: some Scene {
    WindowGroup {
        // 切换注释来查看不同页面
        ContentView()              // 健康 App 复刻页面
        // DesignTokensShowcase()  // 设计令牌展示页面
    }
    .modelContainer(sharedModelContainer)
}
```

### 方式 2：使用 Xcode Preview

1. 打开 `UICopy/DesignTokensShowcase.swift`
2. 点击 Canvas 面板右上角的 Resume 按钮
3. 即可看到实时预览

### 方式 3：创建独立的 Preview Target

在 Xcode 中创建一个新的 App Target 专门用于查看设计系统。

## 📚 设计令牌展示页面包含什么

展示页面分为 **4 个标签页**：

### 1. 颜色标签 🎨
- **文本颜色**：Primary, Secondary, Tertiary, Label 系列
- **背景颜色**：系统背景色、分组背景色
- **系统蓝色**：Primary, Light, Dark
- **健康渐变**：粉橙到蓝色的渐变效果

### 2. 字体标签 📝
- **固定字体系统**：
  - 页面标题 (38pt, Bold)
  - 卡片标题 (20pt, Semibold)
  - 次级标题 (18pt, Semibold)
  - 正文 (17pt, Regular)
  - 说明文字 (15pt, Regular)
  - 按钮文字 (15pt, Semibold)
  - 小文字 (13pt, Regular)
  - 标签文字 (12pt, Regular)

- **动态字体系统**：
  - Large Title, Title, Title 2, Title 3
  - Headline, Body, Callout
  - Subheadline, Footnote
  - Caption, Caption 2

### 3. 卡片标签 📇
展示各种卡片样式：
- **标准卡片**：白色背景、圆角、阴影
- **图标卡片**：左侧大图标
- **横幅卡片**：顶部彩色横幅（通知样式）
- **渐变卡片**：健康 App 渐变背景
- **列表卡片**：分组列表样式

### 4. 组件标签 🧩
展示 UI 组件细节：
- **按钮样式**：主要、次要、彩色按钮
- **圆角对比**：`.continuous` vs `.circular` 对比
- **阴影样式**：Light, Medium, Heavy 阴影对比
- **间距系统**：xs, sm, md, base, lg, xl, xxl 可视化
- **SF Symbols**：常用图标展示

## 💡 使用建议

### 开发时参考
1. 运行 **设计令牌展示页面** 查看所有可用的设计令牌
2. 参考 **iOS系统风格设计指南.md** 了解使用规范
3. 在代码中使用 `DesignTokens.xxx` 引用设计令牌

### 代码示例

```swift
import SwiftUI

struct MyView: View {
    var body: some View {
        VStack(alignment: .leading, spacing: DesignTokens.Spacing.md) {
            Text("标题")
                .font(DesignTokens.Typography.cardTitle)
                .foregroundStyle(DesignTokens.Colors.Text.primary)
            
            Text("描述文字")
                .font(DesignTokens.Typography.caption)
                .foregroundStyle(DesignTokens.Colors.Text.secondary)
            
            Button(action: {}) {
                Text("按钮")
                    .buttonStyle()  // 使用扩展方法
            }
            .buttonStyle(.plain)
        }
        .cardStyle()  // 一键应用卡片样式
    }
}
```
