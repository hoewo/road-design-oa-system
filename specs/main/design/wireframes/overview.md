# 线框图总览

## 文档说明

本文档描述了 道路设计公司项目管理系统 的所有用户界面线框图，覆盖所有关键用户流程和页面状态。

## 界面列表

- **文件**: `01-login.md`
- **平台**: Web（跨平台）
- **用户流程**: 用户通过验证码登录系统（支持邮箱和手机号两种方式）

## 关键用户流程

### 流程1: 用户登录 (P1)
1. 用户进入登录页面
2. 选择登录方式（邮箱/手机号）
3. 输入邮箱或手机号
4. 点击"获取验证码"
5. 输入收到的验证码
6. 点击"登录"
7. 登录成功，跳转到项目列表页面

### 流程2: 创建项目 (P1)
1. 项目管理员进入项目列表页面
2. 点击"创建项目"按钮
3. 填写项目基本信息
4. 配置经营负责人和生产负责人
5. 保存项目信息

### 流程3: 管理项目经营信息 (P1)
1. 经营负责人进入项目详情页面
2. 切换到"经营信息"标签
3. 管理甲方信息、合同、支付、开票等
4. 查看经营信息统计
