## 页面状态说明

每个线框图包含以下状态：
- **正常状态**: 页面正常显示
- **加载状态**: 数据加载中
- **空状态**: 无数据时的显示
- **错误状态**: 错误提示

## 设计说明

- **新线框图**：使用Markdown格式（`.md`），遵循标准模板，包含完整的状态描述和实现注意事项
- **旧线框图**：使用HTML格式（`.html`），便于在浏览器中查看
- 使用简单的边框和占位符来表示UI元素
- 标注了关键交互点和必填字段
- 响应式设计，适配不同屏幕尺寸

## 登录页面改造说明

登录页面已基于NebulaAuth调研结果进行改造：

### 主要变更
- **登录方式**：从用户名+密码改为验证码登录（支持邮箱和手机号）
- **登录流程**：发送验证码 → 输入验证码 → 登录
- **API调用**：直接调用NebulaAuth网关接口，不通过业务服务
- **Token管理**：登录成功后自动存储Token到localStorage

### 新增状态
- 初始状态（选择登录方式）
- 发送验证码中
- 验证码已发送（输入验证码）
- 验证码发送失败
- 验证码错误
- 账号不存在
- 网络错误

### 移除功能
- 用户名密码登录
- 注册功能（系统不提供公开注册）

详细设计请参考：[登录页面线框图](./01-login.md)
