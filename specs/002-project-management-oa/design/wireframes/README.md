## 页面状态说明

每个线框图包含以下状态：
- **正常状态**: 页面正常显示
- **加载状态**: 数据加载中
- **空状态**: 无数据时的显示
- **错误状态**: 错误提示

## 设计说明

- 所有线框图使用HTML格式，便于在浏览器中查看
- 使用简单的边框和占位符来表示UI元素
- 标注了关键交互点和必填字段
- 响应式设计，适配不同屏幕尺寸
