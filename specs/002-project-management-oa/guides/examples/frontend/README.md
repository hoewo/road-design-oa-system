# 前端集成代码示例

本目录包含前端集成相关的代码示例。

## 文件说明

- `api-config.js`: API配置示例
- `auth-service.js`: 认证服务封装示例（登录、注册、刷新Token、登出等）
- `api-client.js`: API客户端封装示例（带自动Token刷新）
- `app-init.js`: 应用初始化示例（页面加载时的认证检查）
- `admin-check.js`: 前端判断当前登录用户是否是管理员的示例

## 使用说明

### 开放注册场景

如果业务方选择开放注册策略，可以使用 `auth-service.js` 中的注册相关函数：
- `sendRegisterCode()`: 发送注册验证码
- `register()`: 用户注册

### 管理员添加场景

如果业务方选择管理员添加策略，前端**不应该**调用注册相关函数，只使用登录功能。
