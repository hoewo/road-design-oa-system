**Service层（业务逻辑）**
```java
@Service
@Transactional
public class ProjectServiceImpl implements ProjectService {
    
    @Autowired
    private ProjectMapper projectMapper;
    
    @Override
    public PageResult<Project> getProjects(int pageNum, int pageSize, ProjectQueryDTO queryDTO) {
        PageHelper.startPage(pageNum, pageSize);
        List<Project> projects = projectMapper.selectByQuery(queryDTO);
        return PageResult.of(projects);
    }
    
    @Override
    public void createProject(ProjectCreateDTO createDTO) {
        Project project = new Project();
        BeanUtils.copyProperties(createDTO, project);
        project.setCreateTime(LocalDateTime.now());
        project.setProjectCode(generateProjectCode());
        projectMapper.insert(project);
    }
}
```

```java
        RedisCacheConfiguration config = RedisCacheConfiguration.defaultCacheConfig()
            .entryTtl(Duration.ofMinutes(30))
            .serializeKeysWith(RedisSerializationContext.SerializationPair.fromSerializer(new StringRedisSerializer()))
```

**查询优化**
```java
// 分页查询优化
@Override
public PageResult<Project> getProjects(ProjectQueryDTO queryDTO) {
    // 使用MyBatis Plus的查询构造器
    LambdaQueryWrapper<Project> wrapper = new LambdaQueryWrapper<>();
    wrapper.eq(Project::getIsDeleted, 0)
           .like(StringUtils.hasText(queryDTO.getProjectName()), 
                 Project::getProjectName, queryDTO.getProjectName())
           .eq(StringUtils.hasText(queryDTO.getStatus()), 
               Project::getProjectStatus, queryDTO.getStatus())
           .orderByDesc(Project::getCreateTime);
    
    IPage<Project> page = new Page<>(queryDTO.getPageNum(), queryDTO.getPageSize());

    return PageResult.of(result.getRecords(), result.getTotal());
}
```
