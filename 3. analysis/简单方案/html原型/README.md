# 道路设计行业OA系统 - HTML原型

## 项目概述

这是基于交互设计线框图开发的HTML原型，完整实现了道路设计行业OA系统的所有核心功能。

## 技术实现

### 前端技术栈
- **HTML5**：语义化标签，表单验证
- **CSS3**：Flexbox布局，Grid布局，动画效果
- **JavaScript ES6+**：模块化编程，事件处理
- **Font Awesome**：图标库

### 设计规范
- **色彩系统**：Element Plus色彩规范
- **字体规范**：系统字体栈
- **间距规范**：8px基础间距
- **组件规范**：统一的按钮、表单、表格样式

### 响应式设计
- **桌面端**：1200px标准宽度
- **平板端**：768px-1024px适配
- **移动端**：<768px提示使用电脑访问

### 2. 本地服务器（推荐）
```bash

# 或使用Node.js
```

## 功能演示

### 登录系统
1. 打开页面后显示登录界面
2. 输入用户名和密码
3. 点击登录按钮进入系统

### 导航使用
1. 顶部导航栏可切换不同功能模块
2. 每个模块都有对应的页面内容
3. 支持页面间的无缝切换

### 数据操作
1. **查看**：点击列表中的"查看"按钮
2. **新建**：点击页面右上角的"新建"按钮
3. **编辑**：点击列表中的"编辑"按钮
4. **删除**：点击删除按钮会弹出确认对话框

### 表单操作
1. 所有表单都有必填字段验证
2. 支持实时表单验证
3. 提交后显示加载状态和结果提示

## 注意事项

1. **数据存储**：当前使用localStorage存储登录状态，刷新页面后需要重新登录
2. **文件上传**：文档上传功能为演示版本，实际文件不会保存
3. **数据持久化**：所有数据操作都是模拟的，不会真正保存到服务器
4. **移动端**：建议在桌面端使用以获得最佳体验
