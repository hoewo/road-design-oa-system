## 访问Swagger UI

每个服务都提供了Swagger UI界面，访问地址：

- **auth-server**: `http://localhost:<port>/swagger/index.html`
- **user-service**: `http://localhost:<port>/swagger/index.html`
- **oauth-server**: `http://localhost:<port>/swagger/index.html`
- **service-registry**: `http://localhost:<port>/swagger/index.html`
- **api-gateway**: `http://localhost:<port>/swagger/index.html`
